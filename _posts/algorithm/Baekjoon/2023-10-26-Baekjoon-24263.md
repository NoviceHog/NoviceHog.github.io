[문제 링크](https://www.acmicpc.net/problem/24263)

# 풀이 과정
```c
MenOfPassion(A[], n) {
    sum <- 0;
    for i <- 1 to n
        sum <- sum + A[i]; # 코드1
    return sum;
}
```

for문이 정직하게 입력값 n에 비례하기 때문에 수행횟수는 n이다.

수행횟수의 최고차항의 차수는 n이기 때문에 1이 정답이 된다.
