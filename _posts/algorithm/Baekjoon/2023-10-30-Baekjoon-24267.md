# 풀이 과정

문제의 for문은 C#코드로 다음과 같이 바꿀 수 있다.
```c#
for (int i = 0; i < n - 2; i++)
{
    for (int j = i + 1; j < n - 1; j++)
    {
        for (int k = j + 1; k < n; k++)
        {
            ...
        }
    }
}
```

문제의 어느정도 규칙성은 보였으나 도저히 식으로 바꿀 수가 없어서
